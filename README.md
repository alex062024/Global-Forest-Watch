# Analyse de la Déforestation - Réserve de Faune à Okapis (RFO)

Ce projet présente une analyse de la perte du couvert forestier et de l'occupation du sol dans la Réserve de Faune à Okapis (RDC) entre 2010 et 2020, basée sur les données de Global Forest Watch et ESA Sentinel-2.
