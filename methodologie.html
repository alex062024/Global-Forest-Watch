Documentation Méthodologique

Processus d'analyse, sources de données et outils technologiques

1. Sources des Données

Cette analyse s'appuie sur trois piliers de données géospatiales et environnementales reconnus mondialement :

  • Global Forest Watch (WRI/UMD) : Utilisation du dataset Tree Cover Loss (Hansen et al.) pour quantifier la perte de forêt primaire annuelle de 2010 à 2020.
  • ESA Climate Change Initiative (CCI) : Données d'occupation du sol (Land Cover) à 300m de résolution pour établir la stratification de référence de 2015.
  • UNESCO World Heritage Centre : Récupération des limites officielles et du statut de conservation du site N° 718.

2. Extraction et Traitement Spatial

Le tracé géographique précis de la Réserve de Faune à Okapis a été extrait via l'API Overpass (OpenStreetMap) en utilisant la relation 401017. Le processus technique comprend :

  1. Récupération des données XML brutes depuis les serveurs OSM.
  2. Conversion du format OSM XML en GeoJSON haute précision pour une intégration web fluide.
  3. Stratification visuelle par classes d'occupation du sol basée sur les statistiques de superficie calculées par l'ICCN et le GFW.

3. Architecture Technologique

Le projet est conçu comme une application web standalone utilisant des technologies modernes :

  • Leaflet.js : Moteur cartographique pour la visualisation spatiale interactive.
  • Chart.js : Génération de graphiques dynamiques pour les séries temporelles et les répartitions proportionnelles.
  • Bootstrap 5 : Framework CSS pour une interface responsive et professionnelle.
  • Python (Pandas/Requests) : Scripts de traitement de données utilisés pour la préparation des datasets.

4. Limites et Précision

Les données de "perte de couvert forestier" indiquent un changement de la canopée mais ne distinguent pas toujours la déforestation anthropique des perturbations naturelles. Toutefois, dans le contexte de la RFO, la corrélation avec l'exploitation minière et l'agriculture itinérante est confirmée par les rapports de terrain de l'ICCN.
